✦ About

Azusa Baileys adalah library WhatsApp API sederhana yang terinspirasi dari karakter
Shirasu Azusa dari game Blue Archive.

Dirancang agar mudah digunakan untuk membuat:
✧ WhatsApp Bot
✧ Automation System
✧ Chat Assistant

---

✦ Installation

npm install github: azusaxaizen/bailzusa

---

✦ Example Usage

const { default: makeWASocket } = require('@whiskeysockets/baileys');

const client = makeWASocket({
  browser: ['Trinity OS', 'Azusa Browser', '1.0.0'],
  printQRInTerminal: true
});
